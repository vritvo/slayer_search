  • Be specific and concise. Make a statement, not a question. Avoid vague or ambiguous queries. ✓ "At Giles's home, Giles and Joyce are on the floor listening to music" ✗ "Where is that scene where Joyce tries to impress Giles?"
  • Search the transcript, not the subtext. This searches actual dialogue and stage directions. Focus on what's explicitly said or shown in the scene. ✓ "Buffy talks about Emily Dickinson" ✗ "Buffy pretends she likes to read poetry"
  • Include the location if you recall it. Locations at the beginning of your query improve results. e.g., "In the car Giles tells Buffy he's proud of her"
  • Include the episode (number and/or name) if you know it. Put it at the start of your query to help improve results. e.g., "2x07 Lie to Me, in the graveyard, Giles tells Buffy about life"
  • "TEXT" → Wrap any text in quotes to force exact substring matching. Only returns results containing that exact string. e.g., Jenny asks Willow if she can be the substitute "teach"er for class → Only returns scenes containing "teach"
  • SPEAKER: → Follow a speaker name with a colon to filter by who's talking. Improves results when you know who speaks. e.g., Buffy: talks to the Master: → Filters to scenes where Buffy and the Master are speakers. Names must match the script exactly.
